# Source Display Frontend — Spec Original

## 기능 개요

AI 답변에 참고한 문서의 출처를 표시하고, 원문을 확인할 수 있다.

## 기능 요구사항

AI 답변 아래에 참고한 문서의 출처 뱃지(파일명, 페이지 번호)를 표시한다.
출처 뱃지를 클릭하면 원문 하이라이트 사이드패널이 열린다.
사이드패널에서 해당 청크가 하이라이트된 원문을 확인할 수 있다.
사이드패널은 닫기 버튼 또는 바깥 영역 클릭으로 닫을 수 있다.
