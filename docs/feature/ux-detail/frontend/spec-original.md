# UX 디테일

## 기능 개요

로딩, 에러, 빈 상태, 다크모드, 반응형 등 전반적인 UX 상태를 처리한다.

## 기능 요구사항

### Skeleton 로딩

파일 업로드 후 인덱싱 중일 때 Skeleton UI를 표시한다.
AI 답변 대기 중일 때 채팅 말풍선 자리에 Skeleton UI를 표시한다.

### 에러 상태

파일 업로드 실패 시 에러 메시지를 표시한다.
API 호출 실패 시 에러 메시지와 재시도 버튼을 표시한다.

### 빈 상태

업로드된 문서가 없을 때 안내 문구와 업로드 유도 UI를 표시한다.
대화 기록이 없을 때 질문 예시를 표시한다.

### 다크모드

시스템 설정에 따라 자동으로 다크모드가 적용된다.
사용자가 직접 토글로 전환할 수 있다.

### 반응형

모바일, 태블릿, 데스크탑 환경에서 레이아웃이 정상적으로 표시된다.

### 비고

다크모드는 Tailwind dark: 클래스로 구현
반응형은 Tailwind 브레이크포인트 (sm, md, lg) 사용
